# Back end

- Node.js server
- Web API
- Routing with Express
- LevelDB database

## Usage

- Start the back end:

  ```bash
  bin/start
  ```

- Start the back end with hot-reload for easier development:

  ```bash
  nodemon --exec "node" bin/start
  ```

- Run the tests with mocha:

  ```bash
  npm run test
  ```

- Init with 3 channels:

  ```bash
  bin/inti
  ```

- Reset the DB:

  ```bash
  bin/resset
  ```
